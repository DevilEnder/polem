# PolemID
Aplicativo desenvolvido para facilitar o gerenciamento de **informações pessoais e médicas** de pessoas com lesão medular.  
O app atua como uma **carteirinha digital**, permitindo o armazenamento de dados de forma organizada, segura e acessível diretamente pelo dispositivo móvel.

---

## 🧩 Funcionalidades

- Cadastro de usuários com dados pessoais e médicos
- Armazenamento de contatos de cuidadores
- Upload e exibição de imagem do usuário
- Integração com **Firebase Firestore** para armazenamento de dados
- Integração com **Firebase Storage** para armazenamento de imagens

---

## 🛠 Tecnologias Utilizadas

| Tecnologia | Descrição |
|-----------|-----------|
| **HTML / CSS / JavaScript** | Estrutura e lógica do app |
| **Cordova** | Build do aplicativo mobile |
| **Firebase** | Firestore + Storage para persistência dos dados |

---

## ⚙️ Como Executar o Projeto

```bash
# clone o repositório
git clone https://github.com/DevilEnder/polem.git

# acesse o diretório
cd polem

# instale o Cordova (caso não tenha)
npm install -g cordova

# adicione a plataforma (exemplo android)
cordova platform add android

# execute o projeto
cordova run android
```

## 🇺🇸 README (English)

# PolemID

PolemID is a digital card application designed to organize and store **personal and medical information** for individuals with spinal cord injury.  
It allows users to keep their data accessible, secure, and easily available through their mobile device.

---

## 🧩 Features

- User registration with personal and medical data
- Caregivers contact information storage
- Image upload and display
- **Firebase Firestore** integration for database storage
- **Firebase Storage** integration for image storage

---

## 🛠 Technologies

| Technology | Description |
|-----------|-------------|
| **HTML / CSS / JavaScript** | App structure and logic |
| **Cordova** | Mobile build and packaging |
| **Firebase** | Firestore + Storage backend |
